== Description ==

The Audience Pixel plugin easily installs and enables the Audience.to tracking code on your Wordpress website.

The tracking code allows you to collect information with Audience.to, on any page of your website to track actions and conversions, build segmented audiences and get rich insights about the people who visits your website. 

The Audience.to tracking code holds a lot of value for your website. The tracking code aids the Audience.to platform to create and deliver personalized messages, content and experiences to your website visitors depending on their behavior. You can retarget users, optimize your ads and messaging or enhance user experience.


== Installation ==

copy the audience folder to /wp-content/plugins/ directory
Activate the plugin through the 'Plugins' menu in WordPress
Goto the Settings > Audience Pixel to change setting

== Frequently Asked Questions ==

= Where can I find my audience app ID? =

You can create a free account at audience.to. The Audience Pixel is the tracking code which is used in this plugin.
